# Deployment Guide

To deploy this app to Vercel immediately:

1.  **Open Terminal** in this folder.
2.  Run the following command:
    ```powershell
    npx vercel
    ```
3.  **Follow the prompts**:
    - Log in (if asked).
    - Set up and deploy: `Y`
    - Which scope: (Select your account)
    - Link to existing project: `N`
    - Project Name: `raja-mantri` (or press Enter)
    - In which directory is your code located: `./` (Press Enter)
    - Want to modify these settings: `N`

4.  **Wait for deployment**. It will give you a `Production: https://...` URL.
5.  **Share that URL** with your friends to play!
